## Problem Scenario
Suppose we have three types of characters in a GameApp:
<ol>
  <li><b>Knight:</b> Attacks with a sword; uses 3 strategies to defend (shield, dodge, magic barrier)</li>
  <li><b>Wizard:</b> Casts spells; uses magic barrier to defend</li>
  <li><b>Archer:</b> Shoots arrows; uses dodge to to defend</li>
</ol>

Implement two types of Strategy:
<ol type="A">
  <li><b>DefenseStrategy</b>
    <ol type="1">
      <li>Shield</li>
      <li>Dodge</li>
      <li>CreateMagic</li>
    </ol>
  </li>
  <li><b>AttackStrategy</b>
    <ol type="1">
      <li>CastSpell</li>
      <li>ShootArrow</li>
      <li>SwingSword</li>
    </ol>
  </li>
</ol>
